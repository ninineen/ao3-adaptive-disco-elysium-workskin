Documentation: All Classes & Elements

All classes go on the elements shown below.

Dialogue Lines — de-convo

Wrap each line of skill dialogue or speech in a p class="de-convo". Uses a hanging indent so the speaker label lines up cleanly.

KIM KITSURAGI — "Detective, I highly advise against drinking from a thermos you found inside a rusted storm drain."

LOGIC [Easy: Success] — This is what a skill line with a passing check looks like.

VOLITION [Medium: Failure] — And this is what a failing check looks like.

HALF LIGHT — A skill line with no check — just commentary or inner voice.

HTML 
<p class="de-convo">
  <span class="de-speaker">KIM KITSURAGI</span> — "Your text here."
</p>

<p class="de-convo">
  <span class="de-skill logic">LOGIC</span>
  <span class="de-check success">[Easy: Success]</span> — Your text here.
</p>

<p class="de-convo">
  <span class="de-skill volition">VOLITION</span>
  <span class="de-check failure">[Medium: Failure]</span> — Your text here.
</p>

<p class="de-convo">
  <span class="de-skill half-light">HALF LIGHT</span> — Your text here.
</p>

Speaker Label — de-speaker

For named characters. Bold, uppercase. Three variants:

KIM KITSURAGI — de-speaker — NPC / named character (neutral gray).

YOU — de-speaker you — Player character (warm off-white).

THE NOTICE BOARD — de-speaker thing — Object, item, or sign (sage green).

HTML 
<span class="de-speaker">KIM KITSURAGI</span>
<span class="de-speaker you">YOU</span>
<span class="de-speaker thing">THE NOTICE BOARD</span>

Player Choices — ol class="de-choices"

Standard choices are orange-red and numbered. Skill checks use a parchment card style (white-check). Add locked to a white-check for an unavailable option. High-stakes checks use red-check.

  1. [Old choice] A previously selected option. Add class="old".
  2. [Selected choice] The option chosen in this scene. Add class="selected".
  3. [Normal choice] An unchosen option. No extra class needed.
  4. RHETORIC [Medium: Difficulty 10] A skill check option. Add class="white-check".
  5. ELECTROCHEMISTRY [Godly: Difficulty 18 — LOCKED] Unavailable. Add both white-check and locked.
  6. HALF LIGHT [Legendary: Difficulty 16] Dangerous check. Add class="red-check".
HTML 
<ol class="de-choices">
  <li class="old">"Text."</li>
  <li class="selected">"Text."</li>
  <li>"Text."</li>
  <li class="white-check"><span class="de-skill rhetoric">RHETORIC</span> [Medium: Difficulty 10] "Text."</li>
  <li class="white-check locked" aria-disabled="true"><span class="de-skill electrochemistry">ELECTROCHEMISTRY</span> [Godly: Difficulty 18 — LOCKED] Text.</li>
  <li class="red-check"><span class="de-skill half-light">HALF LIGHT</span> [Legendary: Difficulty 16] Text.</li>
</ol>

Alert Banners — p class="de-alert [type]"

Two spans inside the p: label first, value second. The check-success and check-failed variants auto-inject dice icons via CSS — the item, money, and task-gained variants inject emoji icons the same way.

Thought Gained thought

Health Healed health-up

Health Damaged health-down

Morale Healed morale-up

Morale Damaged morale-down

Check Passed success

Check Failed failure

Check Passed check-success

Check Failed check-failed

Item Gained item

Money Gained money

Task Gained task-gained

  <span>Thought Gained</span>
  <span>The Cop of the Apocalypse</span>
